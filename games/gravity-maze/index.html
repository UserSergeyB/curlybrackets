About the game

Gravity Maze is designed for quick sessions and careful problem solving. Each level asks you to think about walls, momentum, and the next safe landing before you rotate the maze.

The game is lightweight, readable on mobile screens, and built around short levels that gradually introduce new layouts and constraints.
